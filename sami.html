Guide d'Hébergement sur GitHub Pages

Comment mettre en ligne votre application de calcul de revient

À propos de GitHub Pages

GitHub Pages est un service d'hébergement gratuit qui prend en charge les sites web statiques (HTML, CSS, JavaScript). Il est parfait pour partager votre application avec votre équipe.

1

Créer un compte GitHub

Si vous n'avez pas de compte GitHub :

  1. Allez sur github.com
  2. Cliquez sur "Sign up"
  3. Remplissez les informations requises
  4. Vérifiez votre adresse email
Créer un compte GitHub
2

Créer un nouveau dépôt

Un dépôt (repository) est l'endroit où votre code sera stocké :

  1. Cliquez sur le "+" en haut à droite et sélectionnez "New repository"
  2. Donnez un nom à votre dépôt (ex: "calcul-revient")
  3. Choisissez "Public"
  4. Cochez "Add a README file"
  5. Cliquez sur "Create repository"
3

Téléverser votre code

Ajoutez votre fichier HTML au dépôt :

  1. Dans votre dépôt, cliquez sur "Add file" → "Upload files"
  2. Glissez-déposez votre fichier HTML
  3. Nommez-le index.html pour qu'il soit la page d'accueil
  4. Ajoutez un message de commit
  5. Cliquez sur "Commit changes"
4

Activer GitHub Pages

Configurez GitHub Pages pour votre dépôt :

  1. Allez dans l'onglet "Settings" de votre dépôt
  2. Dans le menu de gauche, cliquez sur "Pages"
  3. Sous "Source", sélectionnez la branche "main"
  4. Laissez le dossier sur "/ (root)"
  5. Cliquez sur "Save"
5

Accéder à votre application

Votre application est maintenant en ligne :

  1. Attendez quelques minutes que GitHub déploie votre site
  2. Rafraîchissez la page des paramètres Pages
  3. Vous verrez l'URL de votre site (sous la forme https://votrenom.github.io/nom-du-depot)
  4. Cliquez sur ce lien pour accéder à votre application
Conseil : Partagez ce lien avec votre équipe pour qu'ils puissent utiliser l'application.

Pour mettre à jour votre application

Si vous devez modifier votre application :

  1. Modifiez votre fichier HTML localement
  2. Retournez sur votre dépôt GitHub
  3. Cliquez à nouveau sur "Add file" → "Upload files"
  4. Téléversez la nouvelle version de votre fichier (cela écrasera l'ancienne version)
  5. Les modifications seront automatiquement déployées sur GitHub Pages

Votre application est maintenant en ligne !

Partagez le lien avec votre équipe pour collaborer.
